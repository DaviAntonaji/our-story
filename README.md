# Nossa História 💕

Uma página especial para guardar e celebrar nossa história de amor.

**Acesse:** [ourstory.antonaji.com.br](https://ourstory.antonaji.com.br/)

## Tecnologias

- React + Vite
- Tailwind CSS
- Framer Motion

## Como rodar localmente

```bash
npm install
npm run dev
```

## Build para produção

```bash
npm run build
```
